# Earnings Reaction Dashboard

Streamlit-based dashboard for analyzing US stock earnings surprises and intraday market reaction trends.

## Features
* **Earnings Surprise Analysis:** Calculates and visualizes the percentage difference between actual EPS and consensus estimates.
* **Intraday Momentum Tracking:** Monitors price changes from announcement through market close.
* **Interactive Filtering:** Allows users to filter by specific stock tickers.

## Technologies Used
* Python
* Streamlit
* Pandas
* Plotly

## How to use
1. Ensure `earnings_data.csv` is updated with the latest ticker data.
2. Run the dashboard locally using: 
   `streamlit run app.py`
